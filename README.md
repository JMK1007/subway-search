# subway-search
 
 HTML,CSS,JavaScripts 연습용 프로젝트
 
 
 지하철 역 이름으로 검색(수도권노선)
 
 (주변 역 정보 오류 있음)
 1. 로고를 누르면 초기화 됩니다.
 2. 역 이름을 입력하고 엔터 또는 검색버튼을 눌러주세요
 3. 환승역의 경우 호선별로 결과를 확인할 수 있습니다.
